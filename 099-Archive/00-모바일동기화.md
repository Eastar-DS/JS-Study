## 추천: 단방향 (fswatch + rsync + launchd)

### 동작 흐름

`[로컬 git repo (SSOT)]
       │ 파일 변경
       ▼
   fswatch 감지
       │
       ▼
   rsync로 iCloud로 복사 (.git 제외)
       │
       ▼
[iCloud Drive 안 vault] ──> [모바일 옵시디언이 보기]`

### 설정 단계 (개요)

**1. 도구 설치**

`brew install fswatch rsync`

(macOS 기본 rsync는 옛 버전이라 brew 권장)

**2. 동기화 스크립트** `~/bin/sync-vault.sh`

`#!/bin/bash
SRC="/Users/eastar/Documents/github/JS-Study/"
DST="$HOME/Library/Mobile Documents/com~apple~CloudDocs/JS-Study-mirror/"

mkdir -p "$DST"
rsync -av --delete \
  --exclude='.git/' \
  --exclude='.obsidian/workspace*' \
  --exclude='.DS_Store' \
  --exclude='node_modules/' \
  "$SRC" "$DST"`

**3. fswatch로 변경 감지 + 자동 실행**

`fswatch -o /Users/eastar/Documents/github/JS-Study | xargs -n1 -I{} ~/bin/sync-vault.sh`

**4. launchd로 백그라운드 상시 실행** (`~/Library/LaunchAgents/com.eastar.vault-sync.plist`)

`<?xml version="1.0" encoding="UTF-8"?>
<plist version="1.0">
<dict>
  <key>Label</key>
  <string>com.eastar.vault-sync</string>
  <key>ProgramArguments</key>
  <array>
    <string>/bin/bash</string>
    <string>-c</string>
    <string>fswatch -o /Users/eastar/Documents/github/JS-Study | xargs -n1 -I{} /Users/eastar/bin/sync-vault.sh</string>
  </array>
  <key>RunAtLoad</key>
  <true/>
  <key>KeepAlive</key>
  <true/>
</dict>
</plist>`

`launchctl load ~/Library/LaunchAgents/com.eastar.vault-sync.plist`

→ 이렇게 하면 **로컬에서 파일 저장 → 1~2초 내 iCloud 반영 → 모바일 옵시디언이 자동 sync**.

**5. 모바일에서 옵시디언 사용**

- iOS 옵시디언 앱 설치
- "Open existing vault" → iCloud Drive 안 `JS-Study-mirror/` 선택

### 주의사항

- `.git/` 제외 필수 (iCloud sync로 corrupt 방지)
- `.obsidian/workspace*`도 제외 (디바이스별 설정 충돌 방지)
- 모바일에서 편집해도 **다음 sync 때 로컬이 덮어씀** (SSOT 정의에 충실)
