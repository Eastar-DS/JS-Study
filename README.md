## 시작하기

이 레포를 Obsidian에서 vault로 열고 [[000-Index/MAIN]]에서 시작한다.

## 폴더 구조

| 카테고리 | 설명 |
|---|---|
| `000-Index/` | 통합 MOC. vault 진입점과 횡단 큐레이션 |
| `099-Archive/` | 옛 통합 일지 등 아카이브 |
| `100-JavaScript/` | JS 언어/런타임 |
| `200-TypeScript/` | TS *(미시작)* |
| `300-React/` | React |
| `400-CS/` | 네트워크, 브라우저, 자료구조 등 CS 기초 |
| 카테고리별 `log.md` | 학습 일지 (시간순) |

## 명명 규칙

- 카테고리 폴더: `<백단위>-<카테고리명>` (예: `100-JavaScript`)
- 세부 영역 폴더: `<일련번호>-<주제>` (예: `101-이벤트루프`)
- 노트 파일: `<순서>-<주제>.md` (예: `01-스레드.md`) 또는 prefix 없음
- 공백 대신 하이픈, 한글 유지
