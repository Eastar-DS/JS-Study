## 큐레이션 (필요 시 추가)

횡단 개념이나 학습 경로가 생기면 이 폴더에 MOC를 추가한다.

예시:
- `면접-준비.md` — 면접 빈출 주제 큐레이션
- `i+1-학습경로.md` — 난이도순 학습 순서 추천
- `비동기-패턴.md` — JS Promise + React Suspense 등 횡단

## 폴더 체계

- 카테고리: 100단위 (100-JS, 200-TS, 300-React, 400-CS, ...)
- 세부 영역: 카테고리 + 일련번호 (101, 102, ..., 401, 402, ...)
- 노트 파일: prefix 없음 또는 영역 내 순서만 (`01-`, `02-`)
